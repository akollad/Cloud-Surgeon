## 🎯 **OPTIMISATIONS COÛTS SPÉCIFIQUES**

### **💡 RAG Intelligence (Économie 60-80%)**
```python
# Recherche similarité avant raisonnement complet
def check_similar_incidents(alert_embedding):
    similar = db.vector_search(alert_embedding, threshold=0.85)
    if similar:
        return similar.resolution  # Coût : ~$0.002 au lieu de $0.012
    else:
        return full_reasoning()    # Coût complet seulement si nécessaire
```

**Impact :** Si 70% des incidents sont similaires → **Économie 8.4¢ par incident**

### **🔄 Prompt Caching (Économie 85%)**
```python
# Cache le contexte système
system_prompt = """
Tu es un agent DevOps expert...
[Contexte de 2000 tokens mis en cache]
"""
# Économie : 85% sur les tokens système répétitifs
```

### **⚡ Tier Flex pour non-critique**
```python
# Incidents non-urgents en mode Flex (-30%)
if incident.priority != "critical":
    service_tier = "flex"  # Économie 30%
```
